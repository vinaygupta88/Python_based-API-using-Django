App Management System

The App Management System is a simple yet powerful web application that allows users to add, retrieve, and delete applications from a database using a user-friendly interface. This system is built using Django with an SQLite database to store app details, including the app name, version, and description.

This API-based system ensures seamless app management and can be easily expanded for more functionalities, such as updating app details or integrating authentication. With a clean and intuitive interface, this tool is useful for developers and administrators who need a lightweight application management solution.

Admin Pannel
  • Add button for add app data to the database
  • Get button for retrieve app details by ID
  • Delete button for delete an app from the database
